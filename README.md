# OHCS E-Library Sponsorship Presentation

A stunning, animated sponsorship presentation for the OHCS E-Library platform - Ghana's AI-powered knowledge platform for civil servants.

## Live Demo

- **Sponsorship Page**: [View Presentation](https://e-library-sponsorship.pages.dev)
- **Full Platform Demo**: [OHCS E-Library](https://ohcs-elibrary.pages.dev)

## Features

- Cinematic hero section with Ghana Black Star animation
- Scroll-triggered Framer Motion animations
- Interactive sponsor tier cards (Platinum, Gold, Silver, Bronze)
- Animated impact metrics counters
- 12-month implementation roadmap
- Ghana flag color theme (Green, Gold, Red, Black)
- Glassmorphic cards with parallax effects
- Mobile responsive design

## Investment Tiers

| Tier | Investment | Key Benefits |
